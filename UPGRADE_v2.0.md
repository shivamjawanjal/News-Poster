## 🚀 How to Use New Features

### Search Posts
```
1. Look for search box (top left)
2. Type: "news", "instagram", "tech"
3. Posts filter automatically
4. Clear box to show all
```

### Use Keyboard Shortcuts
```
1. Try: Ctrl+F (fetch news)
2. Try: Ctrl+P (create posts)
3. Try: F5 (refresh)
4. Try: Ctrl+O (open folder)
```

### Right-Click Menu
```
1. Go to "✨ Posts" tab
2. Right-click on any post
3. Choose action: Preview, Export, Delete, Copy Path
```

### Quick Categories
```
1. Look for buttons below title
2. Click: "Technology", "Business", etc.
3. News fetches for that category
```

### Open Posts Folder
```
Method 1: File > Open Posts Folder
Method 2: Press Ctrl+O
Method 3: Click "📂 Open Folder" button
```

### Copy Post Path
```
1. Select a post
2. Click "📋 Copy Path" button
3. Paste anywhere (Ctrl+V)
```

## 🔧 Quick Setup

### 1. Install Dependencies
```bash
pip install PyQt5 Pillow requests python-dotenv google-generativeai
```

### 2. Setup .env
```bash
# Copy example
cp .env.example .env

# Edit .env with your keys
NEWS_API_KEY=your_newsapi_key
GEMINI_API_KEY=your_gemini_key
```

### 3. Run App
```bash

```
