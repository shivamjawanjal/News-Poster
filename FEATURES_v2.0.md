```python

   🗑️ Delete
```

### 5. Keyboard Shortcuts
```python
Ctrl+F         Fetch News
Ctrl+P         Create Posts
Ctrl+O         Open Posts Folder
Ctrl+,         Settings
Ctrl+Q         Exit
Ctrl+Shift+D   Delete All Posts
F5             Refresh Posts
```

### 6. File Operations
```python
📂 Open Posts Folder
   - Method 1: File > Open Posts Folder
   - Method 2: Press Ctrl+O
   - Method 3: Click folder button
   
📋 Copy File Path
   - Select post
   - Click "Copy Path" button
   - Paste anywhere with Ctrl+V
   
🗑️ Delete Posts
   - Single: Right-click > Delete
   - All: Press Ctrl+Shift+D
   - With confirmation
```

### 7. Professional Menu Bar
```python
📁 File Menu
   - Open Posts Folder

❓ Help Menu

```
